# /copy — Write or review landing page copy

Write or audit copy for Ground Control landing pages.

## Usage
/copy write <section> <page>
/copy review [paste copy to review]

## Brand voice rules (non-negotiable)

### DO
- Short sentences. Declarative.
- Technical terms used correctly and without apology.
- Specific numbers: "checks run every 6 hours" not "frequent monitoring"
- First person for trust: "We store your data in Frankfurt. Not the US."
- Name what competitors don't do (without naming them).
- Questions that the visitor is already asking themselves.
- European directness — say the thing plainly.

### DON'T (flag any of these immediately)
- "Revolutionize" / "transform" / "disrupt"
- "Seamlessly" / "effortlessly" / "easily"
- "Powerful" / "robust" / "cutting-edge" / "next-gen"
- "Game-changing" / "world-class" / "best-in-class"
- "Solution" (use the actual thing it is)
- "Leverage" (use "use")
- Passive voice where active is possible
- More than 20 words in a subheadline
- Exclamation marks (anywhere, ever)

## Copy tone calibration by section

| Section | Tone | Example |
|---------|------|---------|
| Hero | Direct question or bold statement | "Is someone sending emails pretending to be you?" |
| Problem | Stark, factual, no exaggeration | "Your domain is being spoofed right now. You don't know it." |
| How it works | Plain, step-by-step | "Add your domain. We scan continuously. Get alerted." |
| EU trust | Facts only, no adjectives | "Data stored in Frankfurt. Contracts in English and Italian." |
| Pricing | Transparent, no hidden traps | "Free for 3 domains. No credit card. No surprises." |
| FAQ | Honest, direct, answers the real question | [See MXToolbox example above] |
| CTA | Specific action, not generic | "Check your domain free →" not "Get started" |
